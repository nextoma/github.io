# 尼采在世纪的转折点上

## 前言

- 周国平评尼采:
    1. 健全的生命本能 -> 生命哲学
    2. 超越的精神追求 (和 "中庸" 相反) -> 存在主义

- 八十年代的中国 - [尼采, 弗洛伊德, 沙特, 海德格尔] 流行 - [启蒙, 浪漫] (现在回看, 是遥远的传说)

- 此书为什么火了?
    - 尼采一直受全盘否定: "法西斯注意的思想源泉和反动的唯心主义, 唯意志论"
        - 八十年代中期 -> 出现正面评价 => 面目狰狞的政治狂人 -> [真诚思考人生, 个性鲜明] 的哲学家
    - 尼采评价他的时代: [价值崩塌, 信仰空白, 自我迷失, 文化平庸]
    - 此书借尼采的分析 -> 输出了大量周国平的 [感想, 思考] -> 也是中国人面临的问题.

> "他 (尼采) 是一个大学教授, 一个诗人, 一个哲学家, 一个孤独的漂泊者. 他杀死了上帝, 但他不是传说中的那个恶魔. 他是一个真实的人."

## 第一章 我的时代还没有到来

### 世纪末的漂泊者

- 反叛:
    - 生于 7 代牧师世家 -> 大学读神学半年后放弃 -> 专修古典语文学 -> 最终成为基督教死敌.

- 精神危机:
    - 青年时期: 退出团体 -> [离群索居, 冥思苦想, 精神恍惚] -> 使命: 找更真实的人生
    - 被古典文学教授极度认可 -> 曰天才;
    - 研究希腊悲剧起源 -> 拿酒神精神批判 [苏格拉底, 柏拉图] -> 被大众批评

- [叔本华哲学, 瓦格纳的音乐 / 友情] -> 对他影响极深

- 尼采回顾自己的人生经历 1. 合群 (效仿强者) 2. 沙漠 (重估一切价值) 3. 创造 (否定之上 -> 肯定) 不出于权威而出于自己
    - 沙漠: 33 岁左右,[和挚友们决裂, 一再求偶被拒, 健康恶化 => 辞掉当了 10 年的教授] -> 无 [职业, 家庭, 友伴] 孤独漂泊
    - 创造: 我现在勇于追求智慧, 成为哲学家, 而不再只是崇拜他人。

- 矛盾:
    - 作为哲学家 -> 我必须摆脱一切束缚 [职业, 女人, 孩子, 祖国, 信仰] -> 自由, 但作为生命体 -> 我又渴望这些
    - 被文坛和时代抛弃, 他内心矛盾, 渴望友情和爱情, 但也清楚孤独是哲学家的宿命。
    - 人生最后十年精神失常, 神志麻痹, 1900 死, 但 1889 其实已经死了

> 尼采并非一个生性孤僻的人, 年复一年的孤独的漂流也并非一件浪漫的乐事. 在难以忍受的孤寂中, 尼采一次次发出绝望的悲叹:"我期待一个人, 我寻找一个人, 我找到的始终是我自己, 而我不再期待我自己了!"" 现在再没有人爱我了, 我如何还能爱这生命!"< 尼采全集 >, 第 12 卷, 第 324 页." 向我传来的友好的声音如此之少. 如今我孤单极了, 不可思议的孤单…… 成年累月没有振奋人心的事, 没有一丝人间气息, 没有一丁点儿爱."尼采致希德里茨, 1888 年 2 月 12 日. 转引自雅斯贝尔斯: < 尼采导论 >, 第 91 页. 在给妹妹的信中, 他情不自禁地谈到" 那种突然疯狂的时刻, 寂寞的人想要拥抱随便哪个人 "! 尼采致福尔斯特 - 尼采, 1886 年 7 月 8 日. 转引自雅斯贝尔斯: < 尼采导论 >, 第 84 页.

> 可是, 这个害怕孤独, 悲叹孤独的人, 同时又向往孤独, 需要孤独. 因为 "人与人之间的巨大差距迫使我孤独"< 尼采全集 >, 第 12 卷, 第 325 页.; 他感到, 在人群中比独自一人更加孤独. 参看 < 查拉图斯特拉如是说 >: < 归家 >.< 尼采全集 >, 第 6 卷, 第 269 页. 他不肯降格以求, 宁愿走到沙漠里与猛兽一起忍受焦渴, 不愿与肮脏的赶骆驼人同坐在水槽边. 参看 < 查拉图斯特拉如是说 >: < 贱氓 >.< 尼采全集 > 第 6 卷, 第 140 页. 他把孤独当作自己的家, 并且说:"我需要孤独, 就是说我需要恢复, 需要回到我自己, 回到自由的, 轻扬的, 爽朗的空气之呼吸…… 我的一部 < 查拉图斯特拉如是说 >, 就是一曲孤独之颂歌, 或者更明白地说, 一曲纯净之颂歌."< 尼采选集 >, 第 2 卷, 第 413 页.

### 新世纪的早生儿

- 狂妄
    - 自传标题: "我为何如此智慧","我为何如此聪明","我为何写出如此卓越的著作"

- 预言
    - 资本主义的精神危机, 传统价值观念的崩溃, 自然科学发展:
    - 哲学上, 经验主义和理性主义半分天下 -> 根本上动摇基督教信仰 -> 迷信科学, 理性和物质文明
        - 科学有局限性, 单纯物质繁荣造成虚假的幸福 -> 苦闷彷徨, 无所适从
    - 找出路:
        - 左边是 [马克思], 制度 -[劳动分工, 私有制] -> 人的异化 -> 社会革命
        - 右边是 [克尔凯郭尔 - 丹麦 - 宗教, 陀思妥耶夫斯基 - 俄国 - 文学, 尼采 - 德国 - 哲学] -> 三者不相与闻, 得到的见解却一致 -> [诉诸内心, 精神革命]
    - 论思想的丰富性和彻底性, 还是尼采

- 影响:
    1. 说 ["上帝死了","一切价值的重估","现代人无家可归"] -> 人 [存在荒谬性, 自由, 开发性, 无限可能性] 哲学主题讨论
    2. 提倡 [个人至上, 自我实现]
    3. 最早揭示科学局限性, [精神分析学, 现象学运动, 存在主义哲学] 始作俑者
    4. 西方哲学人学主义创始人

- 误解:
    - 生前冷遇, 1888 年 (濒临精神崩溃时) 慢慢被哲学界发现
    - [希特勒常参观尼采博物馆, 纳粹文人颂扬] -> 俨然第三帝国思想先驱
    - 已查明: 尼采妹妹 (反犹太主义者的妻子) 垄断版权 + 遗稿 -> 大量篡改 -> 被误解 -> 被纳粹利用
    - 批判: 尼采的思想中, 确实有糟粕: [对贵族血统的崇拜, 提倡婚配改良种族]

> 尼采渴望被人发现, 可直到 1887 年,"使我异常痛苦的是, 在这十五年里, 仍然没有一个人' 发现 '我, 需要我, 爱我." 尼采致奥维贝克的信. 转引自雅斯贝尔斯: < 尼采导论 >, 第 89 页.

### 误解和发现

- "哲学": 自身是什么, 竟然两千多年来仍然众说纷纭
    - 不同时代有不同的苦恼, 不同哲学家有不同追求
    - 对哲学的看法 -> 对人生的看法 -> 个性
    - 对尼采而言, 哲学是生命

- 世界本质:
    - 质料 (水, 气...), 形式 (数, 存在, 理念)
    - 培根: 解决认识 -> "工具" 是中心任务 => 哲学中心开始向认识论转移
    - 康德: 自觉的把认识能力本身, 作为哲学研究对象 -> 哲学使命为 "看清理性的本质"

- 哲学人学思想
    - 费尔巴哈: 人的价值, 尘世幸福, 热爱人, 热爱自然, 痛恨宗教, 痛恨思辨哲学.
    - 马克思 (宏观社会学): 社会的人的实践本性. 尼采 (微观心理学): 个体的人的非理性本质

- 尼采对苏格拉底的攻击
    - 苏格拉底:
        - 注意力从自然事物 -> 人事, 哲学使命 -> 认识自己
        - 人生问题 -> 道德问题 -> [认识自己 == 关心改善自己的灵魂] -> [道德 = 知识] => [知识 = 美德]
    - 尼采不能容忍 [人生的意义 = 知识 "]
        - "苏格拉底是两千年来欧洲哲学偏离偏离人生根柢的罪魁祸首"
        - 尼采对人生悲观 (受叔本华影响) -> 但不能忍受无意义的人生,
        - 靠艺术赋予人生意义 -> 发展 [酒神精神, 强力意志] -> 提供人生意义解答

- 尼采要哲学迷途知返: 回到根基, 对人生的意义, 提出 [质同, 回答]
    - 智慧!= 知识, 哲学家!= 饱学之士
    - 回响: [海德格尔, 萨特] => 哲学之任务:[揭示个人存在结构 + 意义, 促使个人聚焦体验存在意义]

### 他给西方哲学带来了颤栗

- 康德: 意志为自身立法的公式 -> 唯意志论哲学基础
    - 费希特: 进一步发展 -> 意志 [自我创造世界因果的首要环节, 具有理性性质, 支配思维能动性]
    - 谢林: 进一步强调 -> 除了意志, 别无其他本质 -> 意志即原初

- 尼采哲学, 富有非理性主义色彩 - 否认理性思维可以把握世界本体 - 直观是认识 [绝对, 唯一] 途径
    - 叔本华哲学: 德国古典理性主义 -> 现代非理性主义. -> 取消了康德的 "自在之物" -> 世界为生命意志的客体化, 但仍保留了理性主义色彩
    - 到了尼采, 非理性主义贯穿到哲学的一切领域。

- 尼采哲学 [更少学究气, 触及人生 / 时代的根本], 启发 [哲学, 实证 / 实用 / 弗洛伊德 / 存在主义, 现象学, 生命 / 历史哲学] -> 给西方哲学带来颤栗
    - 尼采首先提出 [存在意义, 自我寻求, 内心充实 / 空虚] -> 存在主义兴起 -> 证明现代意义
    - 尼采探索深层心理 -> 预示精神分析学的建立
        - 弗洛伊德 赞赏 尼采: "比任何人更能深刻认识自己"
        - [阿德勒, 荣格] 借鉴了尼采的 "强力意志" 概念
    - 还影响了 [茨威格，托马斯 · 曼，肖伯纳，黑塞，里尔克，纪德，霍普曼，马尔罗，杰克 · 伦敦，鲁迅......]
        - 斯宾格勒: 尼采提 "重估一切价值" -> 质疑的力量 -> "把尼采的展望变成概观"
    - 尼采对现代哲学家 [法伊欣格尔、西美尔、狄尔泰、克拉盖斯、马克斯 · 舍勒] 等人产生了直接影响
        - 有人还将尼采视为心理社会学的开创者

## 第二章 在人生之画面前

### 哲学和人生

- 现代西方哲学: [宏观社会学, 微观心理学], 都超越了近代仅从 [感性, 理性] 理解人的传统。
    - 马克思: 继承了费尔巴哈的哲学人学 -> 建立历史唯物主义人学 -> "现实的历史的人" 是哲学研究的 [对象, 出发点] -> 哲学使命是人的解放
    - 尼采: 任何哲学都与人的个性不可分, 哲学 == [个性缺陷的补救, 个性丰满的庆祝] -> [发于个性, 体现个性]

- 尼采: 苏格拉底 == 两千年来欧洲哲学偏离人生根本的罪魁祸首:
    - 哲学的注意力 [自然 -> 人事] => [哲学使命 == 认识自己]
    - [人生问题 == 道德问题] => [认识自己 == 关心改善灵魂]
    - 道德是知识 => 知识即美德 =>  人生的意义是知识
    - 不接受人生的意义是知识 => 苏格拉底是罪人

- 认同叔本华的悲观 -> 但不能忍受人生无意义 -> 用 [艺术, 审美] 达人生意义
    - 叔本华: 伟大哲学 => "从生命之画全景中学知生命意义"
    - 后来否定叔本华: 承认自己像 [柏拉图利用苏格拉底] 一样 [利用了叔本华 => 表达思想的工具]
    - 酒神精神 -> 强力意志 (目标: 提供人生意义的答案)

> 哲学关涉人的灵魂, 是对人生意义的不懈寻求。无个性之人不能爱, 离开人生意义的寻求无所谓智慧。智慧不等于知识, 哲学家不等于博学之士。真正的哲人是寻求人生智慧的探索者。尼采认为, 哲学已迷失方向, 应回归自己的根基, 对人生意义提出质问和回答。尼采的呼声在存在主义哲学中激起悠长回响, 海德格尔和萨特的哲学旨在揭示个人存在的结构和意义。

### 首先做一个真实的人

- ` 学者 ` VS ` 哲学家 `, ` 哲学家 ` 不仅思想家, 还为真实的人
    - 科学在自助时伤了仆人 -> 把自己 [冷漠, 干枯] 印 ` 学者 ` 身上了
        - ` 学者 `-[天性扭曲, 冷漠, 创造力]
        - ` 哲学家 `[天性健康, 热情真诚, 创造性]
    - 热爱
        - ` 学者 `: 自命为 ` 爱 ` 知识而求知识 -> "纯粹的求知者" 实为 "精巧的伪善者" (自命清高，无欲地淡视着人生)
        - ` 哲学家 ` 真正的 ` 爱 ` -> 与 ` 死 ` 相伴 -> 愿为所爱者死
    - 初始地看察事物
        - ` 学者 ` 想用 [分析, 了解] 自然 -> 杀死自然
        - ` 哲学家 ` 用 [新生] 自然 -> 丰富自然; 不让 [观念, 意见, 书籍] 挡 [自己, 事物] 间
    - 创作性
        - ` 学者 ` 怕 ` 闲暇 ` -> 埋头书籍 (愚钝式的勤勉) -> 靠别人思想度日
        - ` 哲学家 ` 向往 ` 闲暇 ` -> 自由创造, 玩味自己的思想

- ` 学者 ` 都不思创造性解决问题, 只图证明自己正当
    - ` 秘书型 ` 整理种种材料
    - ` 律师型 ` 辩护被他研究问题的权利
    - ` 牧师型 ` 让人信仰他的信仰
    - ` 犹太学者 ` 用逻辑迫使他人赞同

- 绝对 ` 真诚 `
    - 哲学对象 = 人生, 哲学使命 = 赋予人生意义 -> 关乎人生根本 [乃生死存亡, 非纯学术性]
        - 一般人: 无紧迫感, 认识出于 [利益, 爱好, 无聊, 习惯]
        - 哲学家: 耳旁声音 "认识吧，否则你就灭亡!"
    - 天性健康 + 真诚 => [正确领悟, 忠实探求, 创造意义]
        - 哲学家 -[灵魂, 肉体, 思想] 不可分, 非 [思索蛙, 冷酷观察记录装置]
    - ` 学者 ` 人生态度错误 -[漠视 - 远离 - 虚度]
        - 尼采断定: "` 学者 ` 不能成为 ` 哲学家 `"
    - [真诚人生寻求者 - 可能走错路 - 但态度正确] vs. [学者 - 不犯错误 - 态度错误]
        - 尼采轻蔑: "即使走错路, 仍走在 ` 学者 ` 头顶上"

### 为思想而战

- 哲学家应该是 "使一切崩毁的可怕的炸药"
    - 第欧根尼: "他 (哲学家) 究竟拿出了什么伟大的成果? 学了这么多年哲学，竟没有损人!"
    - 爱默生:"要提防当伟大的上帝让一位思想家到这世界上来的时候，一切东西都有危险了"
    - 哲学: 既定价值 -> [探根究底, 无偏见][估量, 批判]
        - 探求 [新奇, 可疑, 被伦理禁止] -> 追问根据 -> 消除 [掩盖, 歪曲]
        - 用人生尺度衡量 -> 人生真相 -> 决定取舍
    - 许多人 -> 自欺幻想 - 视打破幻想者为仇敌
        - 权势者 -> 培植迷信 - 镇压挑战者
        - 真正哲学家 -> [真实, 坚强] 战士 -> "为你们的思想而战"
    - 哲学堕落 == [依附权势, 迎合迷信] => 赢得名声 - 地位 => "知名的智者" - "媚惑民众的小驴"
        - 瞧不起 [黑格尔, 康德] -> 系统化辩护时代信仰
        - 真正哲学家 == "自由思想者" -> [解剖时代美德 - 戳穿谎言 - 敌对千百年谎骗] -> 悬崖峭壁筑巢的雄鹰 -> [放逐, 寂寞]

- 哲学家 == 精神上的 ` 强者 `
    - [勇敢, 毅力] => [接近 ` 真理 `, 认识 / 肯定现实]
        - 玩弄神圣不可侵犯的事物, 藐视至高无上者
        - 酒神精神: [人生的痛苦 -> 找欢乐, 在沙漠中 -> 孤独的乐趣, 冰窟中 -> 凛冽的快意]
        - 爱仇敌 + 思想的交战 -> 不是真理的垄断者, 否则无聊 -> 追求思想自由, 反对思想独裁
    - ` 弱者 ` -> 生活在 ` 欺骗 ` 中

- [权利, 职业] 败坏了哲学
    - 一旦哲学无利可图 -> 冒牌哲学家争先恐后地抛开哲学 -> 剩下 "一种未经思考过的人生是不值得过的" 的人
    - 哲学为生命 -> 一旦停止人生反思 -> [光阴虚掷, 虽生犹死]

> "哲学家必须是精神上的强者。伟大的思想，与美丽的女子有相同的趣味，决不肯让萎靡的弱者来占有自己"
> "舆论的压迫愈是沉重，自由愈是受到迫害，哲学的尊严就愈是提高"

### 哲学家的命运

- 亚里士多德说：人要独居 -> 野兽或天神
    - 尼采: 哲学家 = 野兽 (桀骜不驯)+ 天神 (充实自足)

### 人生的辩护者

    - `尼采` 否定人生-出世复入世-力图肯定人生

- 叔本华: `个体` 应该: 认清`虚无` -> 自觉否定`生命意志` -> `涅槃`
    - `生命意志`-[世界自在之物, 盲目/不可遏止的冲动] -> 驱动个人产生`欲望`
        - `欲望`-[欠缺,痛苦] => 生命本质:=`痛苦`
        - `欲望`休止 -> `无聊` => 人生:=摇摆[`痛苦`,`无聊`]
    - [`意志`-`无限`,`个人`-`有限`] => `无限` 在 `有限` 无法满足 => `个体`必然死亡

- 尼采-希腊人需要艺术原因: 人生的[痛苦,可悲性] -> 默认前提 -> 艺术:=解救之道
    - [史诗,雕塑] -> [美丽,神圣,光辉] -> 活下去
    - [音乐,悲剧] -> 超脱人生 -> 形而上学的安慰

- 尼采：只有一个世界-非[神圣/道德/理智/仁慈/公正/人道] -> 否定虚假意义-正视现实
    - `科学乐观主义`：`苏格拉底` -> 科学至上-知识万能 -> 浮于表面-不触及人生根柢
    - `基督教乐观主义`：存在`现实世界`外的`真正世界` -> 赋予人生神圣意义 -> 否定现实世界价值
    - `悲观主义`有积极一面：诚实 -> 推翻虚假意义 -> 寻求真实意义(不怀侥幸心理)
        - 但终归消极 -> 败坏了生活的乐趣 -> "死的说教"
            - 人类的难堪: 知道无限 -> 追求永恒 -> 但必死
            - 若被悲观俘虏 -> 时刻感受人生虚无 -> 人类灭亡
        - 叔本华没有自杀 -> 他的理论不彻底 -> 没有观察到自己人生实践中
            - 受叔本华影响的王国维却自杀了
    - 尼采生死观：`死亡` -> `未来`唯一确凿-共同的事情
        - 但人们不愿思考死亡 -> 很好/快乐
        - 健全旺盛的内在生命力 -> 不会屈服于悲观主义 -> 一生抨击叔本华 -> 人生的辩护者
        - 人生[丰富,令人欣羡,神秘] -> 拼命地去感受生命
        - 爱 -> 肯定人生

- 酒神精神实质: 对生命信任消失 -> 不一定忧郁 -> 另一种爱:"爱一个使我们怀疑的女子"
    - `生命` := `女子`[妩媚,无恒, 不驯, 恣肆, 允诺-抗拒, 羞怯-嘲讽, 同情-诱惑]
        - 魅力 =>  受苦 := `快乐` => 恨 := `爱`
